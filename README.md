# JupterNews

Portal de notícias moderno com funcionalidades avançadas.

## Funcionalidades

- Notícias em tempo real
- Sistema de comentários com likes
- Categorias de notícias
- Busca avançada
- Newsletter por email
- Integração com WhatsApp
- AdSense para monetização
- PWA (Progressive Web App)

## Tecnologias

- HTML5
- CSS3
- JavaScript ES6+
- Supabase (backend)
- Service Workers
- Font Awesome
- Google Fonts

## Instalação

1. Clone o repositório
2. Abra `index.html` em um servidor web
3. Configure suas credenciais do Supabase

## Configuração

1. Crie uma conta no [Supabase](https://supabase.com)
2. Crie as tabelas necessárias
3. Atualize as credenciais em `supabase-config.js`
4. Configure seu AdSense
